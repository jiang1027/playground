# Edge Preview Extension

学习和使用Browser Extension API编写Edge浏览器扩展。

## 更新记录

- 2025-10-03: 初始版本。安装后通过links上的右键菜单可打开链接预览

## 安装

1. 下载`edge-preview-extension.zip`文件。
2. 解压缩文件到一个目录。
3. 打开Edge浏览器，进入`edge://extensions/`页面。
4. 打开右上角的“开发者模式”。
5. 点击“加载已解压的扩展”，选择解压缩的目录。
6. 安装完成后，右键点击任意链接，选择“Preview in Sidebar”即可查看预览。
